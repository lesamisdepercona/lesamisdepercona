Fonctionnalité et limite de MongoDB dans la gestion de votre base de données

Assez souvent, nous voyons que le stockage opérationnel principal est utilisé en conjonction avec certains services supplémentaires, par exemple, pour la mise en cache ou la recherche en texte intégral. Une autre approche d’architecture utilisant plusieurs bases de données est celle des microservices, où chaque microservice a sa propre base de données mieux optimisée pour les tâches de ce service particulier. Par exemple, vous pouvez utiliser MySQL pour le stockage principal, Redis et Memcache pour la mise en cache, Elastic Search ou Sphinx natif pour la recherche.
